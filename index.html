Fugle API 設定

前往 developer.fugle.tw 申請免費 API Token，貼入下方儲存。

風險與費用設定
資料管理

所有資料儲存於瀏覽器本機（localStorage），不會上傳任何伺服器。

使用說明
  • 台股輸入代號（如 2330），美股輸入代號（如 AAPL）
  • 台股使用 Fugle 行情 API；美股使用 Yahoo Finance
  • 買入多筆相同股票時，自動計算加權平均成本
  • 停損 / 停利達標時，持股頁面頂端會出現警示
  • 每日點擊「記錄今日資產」可追蹤資產走勢曲線
  • 報價每 5 分鐘自動更新，可手動點擊「更新」
  • 建議定期「匯出 JSON」備份資料
